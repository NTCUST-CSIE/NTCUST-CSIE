# 貢獻者行為準則 (Contributor Covenant Code of Conduct)

## 我們的承諾

為了營造一個開放與熱情的環境，我們作為貢獻者和維護者，承諾參與我們的專案和社群的所有人都能擁有免受騷擾的體驗，無論年齡、體型、身心障礙、族群、性別認同與表現、經驗水平、教育程度、社會經濟地位、國籍、外貌、種族、宗教、或性傾向。

## 我們的標準

有利於創造積極環境的行為包括：

* 使用友善與包容的語言
* 尊重不同的觀點與經驗
* 優雅地接受建設性的批評
* 關注對社群最有利的事物
* 對其他社群成員展現同理心

不可接受的行為包括：

* 使用與性有關的語言或意象，以及不受歡迎的性關注或性暗示
* 挑釁、侮辱或貶損的評論，以及人身或政治攻擊
* 公開或私下的騷擾
* 未經明確許可，發佈他人的私人資訊，例如實體或電子郵件地址
* 其他在專業環境中被合理認為是不適當的行為

## 維護者的責任

專案維護者有權利和責任刪除、編輯或拒絕不符合本行為準則的評論、提交 (commits)、程式碼、Wiki 編輯、問題 (issues) 以及其他貢獻，或暫時、永久禁止任何他們認為其行為不適當、具威脅性、冒犯性或有害的貢獻者。

## 適用範圍

本行為準則適用於專案的所有場域，也適用於個人在公共場域代表專案或其社群時的情況。代表專案或社群的例子包括使用官方專案電子郵件地址、透過官方社群媒體帳號發佈內容，或在線上或實體活動中擔任指定代表。

## 執行

濫用、騷擾或其他不可接受的行為，可透過聯絡專案維護團隊來報告。所有投訴都將被審查和調查，並將產生維護團隊認為對該情況必要且適當的回應。維護團隊有義務對事件的報告者保密。

不遵循或不真誠執行本行為準則的專案維護者，可能會面臨由專案領導層其他成員決定的暫時或永久性的影響。
